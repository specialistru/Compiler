## ✅ Поддерживаемые операции:

| Оператор                        | Описание                  | Приоритет |                  |   |
| ------------------------------- | ------------------------- | --------- | ---------------- | - |
| `OR`, \`                        |                           | \`        | Логическое "или" | 1 |
| `AND`, `&&`                     | Логическое "и"            | 2         |                  |   |
| `=`, `<>`, `<`, `>`, `<=`, `>=` | Сравнения                 | 3         |                  |   |
| `+`, `-`                        | Сложение/вычитание        | 4         |                  |   |
| `*`, `/`, `%`                   | Умножение/деление/остаток | 5         |                  |   |
